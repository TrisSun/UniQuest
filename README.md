# 冒险者公会管理系统（UniQuest）

一个用 C++ 编写的控制台小程序，灵感来自游戏/小说里的「冒险者公会」。面向**江苏省高校学生**：注册后可以**发布委托**、**接取委托**，完成委托能获得经验并**升级**。所有数据保存在本地文本文件中，关闭后再打开数据依然存在。

本项目为《面向对象程序设计语言》课程设计作品。

## 功能特性

- 注册 / 登录（注册时校验学校，仅江苏省高校学生可用，并自选初始称号）
- 称号 / 职业系统（称号即职业，见下文「称号与职业」）
- 发布委托（标题、描述、奖励经验、难度）
- 接取委托（不能接取自己发布的委托，接取时会显示你的称号）
- 提交完成委托，获得经验并自动升级（可连升多级）
- 转职：5 级解锁基础职业，20 级解锁进阶职业
- 浏览委托公告栏、查看个人信息（称号/职业、等级、经验、相关委托）
- 数据用文件持久化保存（`users.txt`、`quests.txt`）

## 项目结构

| 文件 | 说明 |
|------|------|
| `main.cpp` | 程序入口，创建公会对象并启动 |
| `Adventurer.h` / `Adventurer.cpp` | 冒险者（用户）类：账号信息、等级与经验、升级逻辑 |
| `Quest.h` / `Quest.cpp` | 委托类：编号、标题、描述、发布者、接取者、状态、奖励、难度 |
| `Guild.h` / `Guild.cpp` | 公会总控类：文件读写、注册登录、发布/接取/完成委托、菜单交互 |
| `universities.txt` | 江苏省高校白名单，一行一所学校，可直接编辑增删 |
| `UniQuest.sln` / `UniQuest.vcxproj` | Visual Studio 工程文件 |
| `Makefile` | 命令行（g++）编译用 |
| `课设报告.md` | 课程设计文档报告 |

> `users.txt`、`quests.txt` 为程序运行时自动生成的数据文件，首次运行前不存在。

## 编译与运行

### 方式一：Visual Studio（推荐）

1. 双击打开 `UniQuest.sln`
2. 按 `Ctrl + F5`（开始执行但不调试）
3. 自动编译并弹出控制台窗口

### 方式二：命令行（需安装 g++）

```bash
g++ -std=c++11 -o guild main.cpp Adventurer.cpp Quest.cpp Guild.cpp
./guild
```

或在项目目录直接执行：

```bash
make
```

## 使用说明

1. 启动后先在「请先登录」菜单选择 **注册**。
2. 注册时学校**必须填江苏省高校**（如：南京大学、东南大学、南京航空航天大学、苏州大学、河海大学等），否则注册失败。
3. 注册成功后选择 **登录**。
4. 登录后的主菜单：
   - `1` 浏览所有委托
   - `2` 发布委托
   - `3` 接取委托
   - `4` 提交完成委托（完成后获得经验、可能升级）
   - `5` 查看我的信息
   - `6` 退出登录
   - `0` 退出系统

### 简单体验流程

注册两个用户 A、B → A 登录发布委托 → B 登录接取并提交完成 → 查看 B 的等级和经验变化。

## 数据文件格式

- `users.txt`，每行一个用户，空格分隔：

```
用户名 密码 学校 等级 经验 初始称号 职业
```

其中「初始称号」为 `新人` 或 `菜鸟`；「职业」为转职后的职业名，未转职时写作 `-`。（旧版本生成的、缺少这两列的存档仍可正常读取，会按默认值补全。）

- `quests.txt`，每行一条委托，竖线 `|` 分隔（因为标题、描述可能含空格）：

```
编号|标题|描述|发布者|接取者|状态|奖励经验|难度
```

其中状态：`0` 待接取、`1` 进行中、`2` 已完成。

- `universities.txt`，江苏省高校白名单，每行一所学校：

```
南京大学
东南大学
...
```

以 `#` 开头的行视为注释，空行会被忽略。**要新增或删除可注册的高校，直接编辑此文件并重新运行程序即可，无需改动代码、无需重新编译。** 若该文件缺失或为空，程序会自动回退到内置的默认名单。

## 升级规则

每一级升到下一级所需经验为 `当前等级 × 100`。完成委托获得经验后，若经验足够会自动升级，多出的经验保留，一次可连升多级。

## 称号与职业

「称号」和「职业」是同一个东西，会在**接取委托**和**个人信息**处显示。

**未转职时**，称号随等级自动变化：

| 等级 | 称号 |
|------|------|
| 1 ~ 5 级 | 新人 / 菜鸟（注册时自选） |
| 6 ~ 19 级 | 冒险者 |
| 20 级及以上 | 勇者 |

**转职**：达到 5 级后，可在主菜单选择「转职」转入职业。转职后称号即显示为职业名。

- 基础职业（5 级解锁）：`游侠`、`信使`、`赏金猎人`，以及可进阶的 `学者`、`侦探`、`画家`、`好人`
- 进阶职业（20 级解锁，需先是对应基础职业）：
  - `学者` → `贤者`
  - `侦探` → `天眼`
  - `画家` → `吟游诗人`
  - `好人` → `君子`

`游侠`、`信使`、`赏金猎人` 没有进阶职业。

## 已知不足

- 密码以明文保存，安全性较低
- 用户/委托查找为线性遍历，数据量大时效率一般
- 委托标题、描述中不能包含竖线 `|`（会破坏文件格式）

后续可扩展：委托搜索分类、积分排行榜、管理员审核等。
